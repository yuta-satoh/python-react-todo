## 仮想環境作成
```
python3 -m venv .venv
```

## 仮想環境に入る
### Mac
```
source .venv/bin/activate
```
### Win
```
.venv\Scripts\activate.bat
```

## ライブラリ読み込み
```
python -m pip install -r requirements.txt
```

## Pythonプロジェクト起動
```
python app.py
```

## Reactプロジェクト起動
```
cd todo/
npm start
```
